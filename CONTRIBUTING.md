## Contributing

Thanks for your interest in contributing! To keep contributions easy to review:

- Fork the repository and create a feature branch for your work.
- Keep changes focused and self-contained.
- Add or update tests where applicable.
- Run `python main.py` locally to validate runtime behavior.
- Open a pull request describing the change and why it's needed.

For changes that touch API keys or secrets, do not include real keys; reference
`config/api_keys.example.json` instead.
